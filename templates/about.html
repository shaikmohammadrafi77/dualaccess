About Dual Access Cloud

Advanced Secure File Management System

🚀 Project Overview

Dual Access Cloud is a cutting-edge cloud-based file management system that implements advanced security protocols and dual access control mechanisms. Built with modern web technologies, it provides a secure, scalable, and user-friendly platform for managing sensitive files in a cloud environment.

Our system ensures that your files are protected with military-grade encryption while providing flexible access control through role-based permissions and attribute-based encryption.

🛠️ Technologies Used

Python Flask

Modern web framework for building secure and scalable web applications

SQLAlchemy

Powerful ORM for database management and secure data operations

CP-ABE Encryption

Ciphertext-Policy Attribute-Based Encryption for advanced security

HTML5 & CSS3

Modern frontend technologies with responsive design

Bootstrap

Responsive framework for beautiful and mobile-friendly interfaces

Cloud Storage

Secure cloud-based file storage with encryption at rest

✨ Key Features

Role-Based Access Control

Four distinct user roles (Owner, End User, Authority, Admin) with granular permission controls

Advanced Key Management

Automated key request and approval system with secure key distribution

Secure File Upload

Encrypted file upload with automatic encryption and secure cloud storage

Admin Dashboard

Comprehensive admin panel for system oversight and user management

File Deletion Control

Secure file deletion with owner-only access and automatic cleanup

Responsive Design

Mobile-friendly interface that works perfectly on all devices
